Student Name: Baasim Hussain
Dev Tools: Python, PySide6, PyCharm

How To Run:
1. Enter the unit in the top right text box, either using a whole or decimal number
2. Choose the conversion, either inches to meters or meters to inches
3. Click the convert button at the bottom, and text will pop up showing the original number and the converted unit depending on which unit was to be converted
4. Click the clear button to clear the text box unit and the converted units above the buttons
5. To quit, simply press the exit button
